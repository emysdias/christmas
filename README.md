## ✔️ Pré requisitos

- Node/ npm.
- Docker / docker-compose

## 👉 Como rodar

Para clonar o repositório

```bash
$ git clone https://github.com/emysdias/christmas.git
```

Entre no repositório e instale as dependências

```bash
$ docker-compose up --build
```

Aproveite o jogo!
